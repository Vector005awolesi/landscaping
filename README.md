# Vera Landscaping Website

A modern, responsive website for **Vera Landscaping**, built to showcase professional landscaping services, completed projects, and provide an easy way for clients to get in touch.

---

## Overview

The Vera Landscaping website presents the company’s services, portfolio, and contact information in a clean, user-friendly layout. It is designed to reflect professionalism, trust, and quality workmanship.

The website focuses on:
- Highlighting landscaping services
- Showcasing completed landscaping projects
- Making it easy for clients to request information or quotes

---

## Key Features

- **Responsive Design**  
  Optimized for desktop, tablet, and mobile devices.

- **Service Pages**  
  Dedicated pages for landscaping services including garden design, lawn care, irrigation, and tree services.

- **Portfolio Gallery**  
  A visual gallery of completed landscaping projects with category filtering and project previews.

- **Interactive Elements**  
  Subtle animations, hover effects, and modals for a polished user experience.

- **Contact Form**  
  Front-end validated contact form for customer inquiries.

---

## Technology Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Bootstrap 5  
- Font Awesome
